# Digital Ream

A single self-contained web viewer for *Digital Ream* by Lizabetta Frankline (501 pages).

## What's here
- `index.html` — the entire viewer (HTML/CSS/JS + all page data embedded). No build step, no dependencies.

## Hosting on GitHub Pages
1. Create a new GitHub repo named `digital-ream` (or `Digital Ream`).
2. Upload `index.html` to the repo root (GitHub web UI: **Add file → Upload files**).
3. Go to **Settings → Pages**, set source to the `main` branch, root folder.
4. GitHub will publish it at `https://<your-username>.github.io/digital-ream/`.

## Local preview
Just open `index.html` in any browser — no server required.
